Overview

The Internal data source is used to monitor various performance aspects of Mango Automation.

Some things, like Persistent TCP data sources and any polling data source, will create new monitors which the internal data source can record values from. Because this is dynamic, the internal data source can automatically create points for new monitor IDs that match the Create points pattern. If no pattern is supplied, no points are automatically created. If you wish to use more than one of the supplied patterns, you may conjoin them as

(?:pattern|pattern)
