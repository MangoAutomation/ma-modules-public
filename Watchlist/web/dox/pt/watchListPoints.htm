Esta seção provê uma lista de todos os data points do sistema em que você possui acesso. Seu acesso depende das suas configurações na área de usuário (). Fale com seu administrador caso você não tenha acesso a todos os data points nos quais gostaria.Se você não possui data points cadastrados, pode criá-los cadastrando (inicialmente) data sources. ().

Os data points são organizados em uma hierarquia (árvore), que é configurada na área de Hierarquia. (). A hierarquia é inteiramente arbitrária e é compartilhada por todos os usuários. Você precisa ser um administrador para configurá-la.

Para adicionar um data point na hierarquia para sua atual watch list, clique em seu nome. O ícone da flecha () ficará desabilitado, indicando que o ponto já se encontra em sua watch list.
