This data source reads a file as an ASCII character stream on a line by line basis.

Configuration

Every data source requires a Name, which can be any description, and an XID, which must be unique among all data sources. The Update period specifies how frequently the data source checks the file. The File Path points to the file you want to access information from. The file path is absolute, but networked paths are not supported. File paths can be restricted in the ASCII File section of the system settings. Use the "Check" button to verify your file path references an existing file.
