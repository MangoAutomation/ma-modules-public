The Point Identifier is an ASCII string that will be used to identify a line in a file as corresponding to this data point. It is compared to the captured Regex value using String.equals.

The Point Identifier Index is the integer index for the capturing group that is the Point Identifier.

The Value Index is the integer index for the capturing group that will contain the new value.

The Value Regex is a Java-style regular expression which will match the (identifier)(value) pair. Syntax, tutorials, and classes can be found at the Oracle website, here.

Timestamp patterns are defined in the documentation for the SimpleDateFormat class, which can be found here. When importing with timestamps, the protections on times for new point values are disabled.
