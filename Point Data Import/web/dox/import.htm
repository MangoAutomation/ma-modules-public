Data Import Help

Historical point values can be imported from other Mango instances or backups from this system. The import format is exactly the same as the export format.
On the import data page first select the type of data to import. CSV format or XLSX format.

XLSX Data Import

Data can be exported from 2 areas for backup or to prepare data to be modified. It is highly recommeded to export some data (or use the below file) and modify the file instead of hand writing a file for import.
  • The point details page can export an XLSX file from the chart area.
  • The report modules can export data as an XLSX file.

Data Format

The import file should have the following column headers
  • Point Value ID - the id of the point value in the database, setting to -1 will create a new point value otherwise will overwrite existing value
  • Data Point XID - the Data Point XID that this value is be assigned to
  • Device Name - for display only, this vlaue is not saved
  • Point Name - for display only, this value is not saved
  • Time - the time for the point value
  • Value - the value that will be saved into the system
  • Rendered - for display only, this value is not saved
  • Annotation - the annotation to match to the value

Data Format

The import file should have the following format, not limited to 2 data points or 2 data values per point.
-empty-Point1 XIDPoint2 XID
1970/1/01 12:00:001.01.1
1970/1/01 12:01:002.02.1

In the above table the values of 1.0 and 2.0 will be assigned to Point 1 and 1.1 and 2.1 will be assigned to Point 2.
