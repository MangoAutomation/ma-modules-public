Configuration

  • Data type - the matched value is attempted to be converted from its string value to this (see below)
  • Point identifier - the String that is used to determine if the message in the data source is for this point. It is used to check for an exact match to the data source's point identifier group
  • Value group - the matching group from the point's value regex that will be used to extract a value for this point
  • Value regex - the regular expression pattern to use on the entire message to extract a value.

Data Type Conversion

  • Alphanumeric values are straight from UTF-8 bytes
  • Binary for hex representations 0 is false, 1 is true. For non-hex case insensitive 'true' or 'false' strings.
  • Multistate for hex the value of 2 or 4 bytes is converted to an integer. For non-hex strings of '124' are converted to the integer 124.
  • Numeric for hex the value of 4(float) or 8(double) bytes is converted to a floating point precision number. For non-hex strings of '1234.3' are converted to the floating point value 1234.3
